# ChatBotX

本项目是基于50w小黄鸡对话语料构建的Transformer生成式单轮对话模型。本项目受启发于另外一个使用seq2seq模型构建的单轮对话模型。（项目地址：https://github.com/Schellings/Seq2SeqModel）

目前模型的效果很差，待进一步完善...
